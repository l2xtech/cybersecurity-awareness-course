Module 8: Document and File Security

Learn how to manage and share documents securely

Learning Objectives

  • Manage documents securely
  • Implement proper file access controls
  • Securely share and dispose of documents
  • Recognize sensitive documents

Introduction

Documents and files contain valuable information. This module teaches how to manage, share, and dispose of them securely to protect intellectual property and comply with regulations.

Document Classification

Categories:

  • Public: Freely shared (e.g., marketing).
  • Internal: Employees only (e.g., announcements).
  • Confidential: Sensitive with limited access (e.g., HR files).
  • Restricted: Highly sensitive (e.g., IP, financials).

Methods:

  • Visual markings (e.g., headers, watermarks)
  • Metadata tags
  • Color coding
  • File naming conventions

Secure File Storage & Sharing

Storage:

  • Company servers and document systems
  • Cloud with encryption
  • Organized folders, access control
  • Backups and clean desk policies

Sharing:

  • Use approved tools (e.g., SharePoint, Google Workspace)
  • Expiring links, password-protection, encryption

Digital Rights Management (DRM)

  • Control access and actions (e.g., edit, print)
  • Set expiration and revoke access
  • Use watermarks and usage tracking

Secure Document Disposal

Physical: Shred, incinerate, or use certified services.

Digital: Overwrite with secure deletion tools, sanitize devices.

Document: Track and log disposal for compliance.

Preventing Data Leakage

Risks: Accidental email, printers, unsecure sharing, metadata.

Protection:

  • Use DLP tools, monitoring, user education
  • Password-protect, redact, remove hidden content

Quick Tips ✅

  • ✅ Check classification before sharing
  • ✅ Use approved sharing platforms
  • ✅ Remove metadata
  • ✅ Shred sensitive physical docs
  • ✅ Lock your screen when away

Summary

  • Classify and store documents correctly
  • Control and monitor access
  • Dispose securely
  • Prevent leaks and follow security policy
