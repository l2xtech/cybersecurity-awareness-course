How Our Training Aligns with ISO 27001

The cybersecurity training modules you've completed are designed to align with ISO 27001 principles and requirements. Here's how each module connects to the standard:

Training Module ISO 27001 Alignment Practical Application
Module 1: Introduction to Information Security Foundation for understanding information security importance Recognizing the value of information and the need to protect it
Module 2: Common Cyber Threats Risk identification and threat awareness (A.8.2) Identifying potential security threats in daily activities
Module 3: Password Security Access control and authentication (A.9.4) Creating and managing strong passwords
Module 4: Email Security Information transfer and communication (A.13.2) Recognizing phishing and handling email securely
Module 5: Safe Internet Browsing Protection from malware (A.12.2) Safe browsing habits and malware prevention
Module 6: Mobile Device Security Mobile device policy (A.6.2) Securing mobile devices and remote work
Module 7: Data Protection Information classification (A.8.2) Handling sensitive data appropriately
Module 8: Document Security Media handling (A.8.3) Secure document management and disposal
Module 9: Security Policies Information security policies (A.5.1) Understanding and following company policies
Module 10: Incident Reporting Information security incident management (A.16) Reporting and responding to security incidents
